# 📚 Ders Sayacı

## 🚀 Projenin Tanıtımı

Bu proje öğrencilerin günlük ders çalışma sürelerini takip edebilmesi için geliştirilmiştir.

AntiGravity AI teknolojisi kullanılarak öğrencinin çalışma süresi analiz edilir ve yapay zekâ destekli öneriler sunulur.
